# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities privately by opening a
[security advisory](https://github.com/wingnut128/akvutil/security/advisories/new).

Do **not** open a public issue for security reports.

You can expect an initial response within a few days. If the issue is
confirmed, we will coordinate a fix and a disclosure timeline with you.

## Supported Versions

Security fixes land on `main` and ship in the next tagged release. Only the
**latest release** (and the current `main`) is supported — there are no
backports to older tags. If you run a prebuilt binary from the
[Releases](https://github.com/wingnut128/akvutil/releases) page, update to the
latest release to receive fixes.
